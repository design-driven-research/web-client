# RDD - Web client

The ultimate recipe builder and coster

# Dev setup

1. Clojure tool.deps
1. Java
1. Calva
1. Shadow CLJS

Install deps

```
yarn
```

Start dev env. This will fire up shadow-cljs and tailwind via postcss

```bash
yarn dev
```

Navigate to: [http://localhost:4200](http://localhost:4200)
